# PIXI Application

![workflow](https://github.com/abodelot/jquery.json-viewer/actions/workflows/node.js.yml/badge.svg)

## Setup

Install dependencies:

    yarn install

Start development server:

    yarn run serve

Run eslint:

    yarn run lint

## Deployment

Build for production:

    yarn run build

Serve `dist` directory:

    node server.js
